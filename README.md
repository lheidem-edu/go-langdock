# go-langdock

API library for interacting with the [Langdock API](https://docs.langdock.com/api-endpoints/api-introduction) - written in Go.

> [!NOTE]
> This library is a work in progress and is primarily used for the langdock-knowledge-sync project, so it only includes the features required for that use case.

## Installation

```bash
go get github.com/lheidem-edu/go-langdock@latest
```
